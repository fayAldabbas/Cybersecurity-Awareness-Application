# Cybersecurity Awareness Application

## Overview
An educational mobile application designed to teach children cybersecurity concepts.

## Features
- Interactive scenarios  
- Quiz-based learning  
- Gamified experience  

## Topics Covered
- Phishing  
- Privacy  
- Safe Internet Usage  
- IoT Security  

## Key Learning
- Cybersecurity awareness design  
- User experience for children  
- Educational content development
